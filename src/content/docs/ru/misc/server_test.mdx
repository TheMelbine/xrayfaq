### 2. YABS — комплексный тест сервера (CPU, диск, сеть)

Скрипт автоматизирует запуск популярных бенчмарков для комплексной проверки сервера: диск (fio), сеть (iperf3), CPU/память (Geekbench).

```bash
bash <(curl -sL yabs.sh) -4
```
